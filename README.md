# Semantic Function Checker

A desktop application for **semantic analysis of functions**. This tool helps detect semantic errors in function definitions and calls, ensuring that your code follows proper rules before execution.

## Features

- Check semantic correctness of function declarations and calls.
- Detect mismatched argument counts and invalid function usage.
- Provides detailed error messages for debugging.
- User-friendly desktop interface.
